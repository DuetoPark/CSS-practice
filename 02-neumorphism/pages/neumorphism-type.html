볼록버튼 Convex Button

완성본

주로 사각형과 원형입니다. 사각형의 뾰족한 꼭지점은 둥글게 깎여있습니다. 납작한 버튼이 앞으로 튀어나온 형태이며, 튀어나온 면은 평평합니다. 노트북의 키보드와 유사합니다. 그림자효과가 있어서 입체적으로 인식됩니다. 그림자는 CSS의 박스섀도우를 이용했습니다.
뉴모피즘을 구현하는 CSS코드 입니다. 제작방법이 궁금하신 분들은 다음으로 넘어가주세요. 
                
                  .convex-button{
                    display: block;
                    width: 200px;
                    height: 200px;
                    border-radius: 40px;
                    background-color: #F0F0F3;
                    color: #B1B5C6;
                    font-size: 18px;
                    box-shadow: -5px 5px 10px rgba(216, 216, 219, 0.2),
                     5px -5px 10px rgba(216, 216, 219, 0.2),
                     -5px -5px 10px rgba(255, 255, 255, 0.9),
                     5px 5px 13px rgba(216, 216, 219, 0.9),
                     inset 1px 1px 2px rgba(255, 255, 255, 0.3),
                     inset -1px -1px 2px rgba(216, 216, 219, 0.5);
                  }
