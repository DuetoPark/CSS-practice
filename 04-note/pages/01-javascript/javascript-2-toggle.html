🙋토글 (Toggle)🙋

👶토글하는 방법

🌚HTML

꼭 지켜야 하는게 있다면, 태그에 클래스 선언할 때 규칙을 세우는 것.

🌞CSS

웹 페이지를 만들 때, 반복되는 스타일은 별개의 파일을 만들어서 그 파일에 작성하고 관리한다.(나는 common.css에 작성)
토글되었을 때 보여질 스타일도 common.css 파일에 작성 

              
                .background-toggled{
                  background-color: #525252;
                  color: #FFFFFF;
                }

🌝Javascript

작성한 스타일을 추가하는 함수(toggled();)를 만든다. 핸들러(toggleHandler)에 함수(toggled();)를 선언한다. 이벤트를 줄 요소에 addEventListener 메소드를 선언한다. 

              
                function toggled () {
                  document.body.classList.add('background-toggled');
                }

                let toggleHandler = function () {
                  toggled();
                }

                toggleButton.addEventListener('click', toggleHandler);

🎉나 사실 토글 잘 못한다!!!

📀내가 토글을 잘 못했던 이유

  • common.css를 만들지 않았다
  • 클래스명이 자유분방했다
  • 어떻게하면 원하는 DOM 요소만 선택할 수 있는지 고민하지 않았다

📢그래서 어떻게 했냐면

  • common 스타일을 작성했다
  • 클래스명에 통일감/규칙을 세웠다
  • 대입연산자를 사용해서 DOM 요소를 선택했다

🎬토글을 연습하면서 배운 점

  • classList.add()는 진짜 편하다
  • 현재 상태를 저장할 변수가 필요하다
  • 부모요소에 이벤트를 지정하면 반복문의 사용을 줄일 수 있다
  • 두렵다고 피하면 그 상황이 오래갈 뿐이다. 조금 망하면 어때 라는 마음가짐으로 몸통박치기를 해보자. 홀가분하당
