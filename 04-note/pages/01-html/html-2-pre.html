🔍pre 태그의 정의

pre(Preformatted Text) 태그란?
data에 대한 정보를 나타내는 태그이다.
우리가 만든 서비스를 최대한 노출 시키고 싶을 때 이용하는 태그이다.
pre 태그의 위치
html파일의 <head> 태그 안에 들어간다.

👾pre태그의 자식태그에는 뭐가 있을까?👾

code
Defines a piece of computer code
samp
Defines sample output from a computer program
kbd
Defines keyboard input
var
Defines a variable
