⚾글자 안에 이미지 넣는 방법

그림 위에 글자를 배치하기는 많이 해봤지만, 글자 안에 이미지를 배치하는 건 안해봤다. 어.. 어떻게 해야하는지 구상을 해봤는데, background image와 text의 color를 이용하면 되지 않을까? 아니었다. background-clip, -webkit-background-clip, color 세가지 프로퍼티를 이용한다. 

              
                background-clip: text;
                -webkit-background-clip: text;
                color: transparent;
종류 border-box 배경이 테두리의 바깥 경계까지 차지합니다. (Z축 순서 상 테두리 아래 위치) padding-box 배경이 안쪽 여백의 바깥 경계까지 차지합니다. 테두리 밑에는 배경을 그리지 않습니다. content-box 배경을 콘텐츠 상자에 맞춰 그립니다. text 배경을 전경 텍스트 위에만 그립니다.
