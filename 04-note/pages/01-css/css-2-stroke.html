🍆텍스트 외곽선 넣는 방법

css를 사용하여 텍스트에 외곽선, stroke 효과를 주는 방법 외곽선은 텍스트를 강조하는 효과적인 방법 

            
              -webkit-text-fill-color: white;//투명도 가능
              -webkit-text-stroke: 1px #000;
대신, 표준방법이 아니라서 웹킷 및 모질라 계열의 브라우저에서만 적용된다. 웹 크로스 브아우징 방법 text-shadow를 이용한다. 
              
                .text-stroke {
                  color: #f00;
                  text-shadow:
                  -1px -1px 0 #000,
                  1px -1px 0 #000,
                  -1px 1px 0 #000,
                  1px 1px 0 #000;
                }
애니메이션 설정하면, 색만 바뀌고 두께는 안바뀐다. 
              
                /* Only the color will change, not the width */
                @keyframes colorchange {
                  0% {
                    -webkit-text-stroke: 10px red;
                  }
                  100% {
                    -webkit-text-stroke: 20px green;
                  }
                }
