😈word-wrap😈

👻줄바꿈 속성

word-wrap 속성은 긴 텍스트를 강제로 끊어 줄바꿈을 해주는 속성이다.
overflow 속성과 비슷하다.

word-wrap는 주소같은 문자열을 끊을 때 사용한다.
(참고, 인터넷 주소는 줄바꿈이 되지 않는다.)

주로 비아시아 언어의 줄바꿈을 제어한다.
아시아언어(한국어, 중국어, 일본어 등등)의 줄바꿈은 word-break속성을 이용한다.

✏️word-wrap 속성값

normal
기본값, 글자가 길어도 끊어지지 않고 한 줄에 계속 표시
break-word
강제로 끊어서 줄바꿈 함
부모요소로부터 자식요소에게 상속된다.

✏️word-wrap 이 가능한 요소

  • 인라인 요소
  • 블록요소
  • 절대적 위치(position: absolute;)의 블록요소

✏️word-break

아시아, 비아시아 언어의 줄바꿈 모두를 제어한다.

word-break 속성

normal
기본값으로, 일반적인 규칙에 따라 줄바꿈 함.
break-all
문자를 강제로 줄바꿈.
아시아 언어에서 사용할 땐 normal속성처럼 행동하며,
비아시아 언어에서 사용할 땐 임의대로 줄바꿈 함.
keep-all
아시아 언어에서 사용할 땐 normal속성처럼 행동하며,
문자 쌍 사이에서 줄바꿈을 금지함.

word-break과 word-wrap의 차이

word-break
가차없이 단어를 중간에 잘라버림
word-wrap
단어를 자르진 않음
