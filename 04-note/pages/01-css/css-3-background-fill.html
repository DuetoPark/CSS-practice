⚾글자 안에 이미지 넣는 방법

그림 위에 글자를 배치하기는 많이 해봤지만,
글자 안에 이미지를 배치하는 건 안해봤다.
어.. 어떻게 해야하는지 구상을 해봤는데,
background image와 text의 color를 이용하면 되지 않을까?

아니었다.
background-clip, -webkit-background-clip, color 세가지 프로퍼티를 이용한다.

            
              background-clip: text;
              -webkit-background-clip: text;
              color: transparent;

background-clip의 value

  1. border-box
  2. padding-box
  3. content-box
  4. text
